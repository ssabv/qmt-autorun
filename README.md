# QMT 自动登录工具

定时启动国金 QMT 并自动填写账号密码登录，支持**普通电脑**和**云服务器**。

## 功能

- 图形界面配置账号、密码、程序路径
- 周一至周五 8:00 自动启动并登录
- 一键手动启动测试
- 打包后无需 Python 环境

## 云服务器用户

云服务器用户需要额外设置：

1. **必须使用 ToDesk、UU 远程或其他类似的远程软件**
   - ❌ 不能使用 Windows 自带的远程桌面（RDP），RDP 断开后会话会挂起
   - ✅ ToDesk/UU 等软件支持「无人值守模式」，断开后会话保持活跃

2. **关闭锁屏和睡眠**
   - 详见下方「服务器设置教程」

普通电脑用户直接运行即可，无需任何特殊设置。

## 服务器设置教程（仅云服务器用户）

> 普通电脑用户跳过此章节，直接运行即可。

### 必须用 ToDesk/UU 而非远程桌面

云服务器上安装 ToDesk 或 UU远程，开启「开机自启」和「无人值守访问」。

以后远程控制必须用这些软件，**不要用 Windows 远程桌面**。

### 关闭锁屏+睡眠（按顺序做）

#### 一、关闭屏幕保护 + 取消密码

1. 右键桌面 → 个性化 → 锁屏界面 → 右下角 屏幕保护程序设置
2. 屏幕保护选「无」，取消勾选「在恢复时显示登录屏幕（密码保护）」→ 确定

#### 二、电源计划：显示/睡眠设为「从不」

1. 控制面板 → 硬件和声音 → 电源选项
2. 当前电源计划 → 更改计划设置：
   - 关闭显示器：从不
   - 使计算机进入睡眠状态：从不
3. 点「更改高级电源设置」：
   - 显示 → 关闭显示时间：0（从不）
   - 睡眠 → 睡眠、休眠：禁用/从不
   - 应用 → 确定

#### 三、组策略：彻底禁用锁屏

1. Win+R 输入 `gpedit.msc` 回车
2. 定位：`计算机配置 → 管理模板 → 控制面板 → 个性化`
3. 右侧双击「不显示锁屏」→ 设为「已启用」→ 应用

#### 四、注册表（无组策略版本/家庭版用）

1. Win+R 输入 `regedit` 回车
2. 定位：`HKEY_LOCAL_MACHINE\SOFTWARE\Policies\Microsoft\Windows`
3. 右键 Windows → 新建 → 项，命名 `Personalization`
4. 选中 Personalization → 右键新建 → DWORD(32位)值，命名 `NoLockScreen`，值设为 1 → 确定

#### 五、远程桌面额外设置

1. 打开「远程桌面连接」→ 显示选项 → 高级 → 勾选「保持连接」
2. 服务器本地：运行 `secpol.msc` → 本地策略 → 安全选项 → 交互式登录：不显示上次的用户名 → 设为「已禁用」

#### 六、防止账号被锁定

1. `gpedit.msc` → 计算机配置 → Windows 设置 → 安全设置 → 账户策略 → 账户锁定策略 → 账户锁定阈值 → 设为「0」（不锁定）

**做完上面步骤后，重启一次服务器。**

## 运行

```bash
pip install -r requirements.txt
python main.py
```

## 打包为 exe

```bash
pip install pyinstaller
pyinstaller --onefile --noconsole main.py
```

## 配置文件

首次运行后会生成 `config.json`，包含你的账号密码信息。**此文件不会上传到 GitHub。**

## 注意事项

- config.json 包含账号密码，请妥善保管
- 本工具仅为自动化操作便利，不涉及任何交易接口
- 使用前请确保服务器设置正确（见上方教程）
